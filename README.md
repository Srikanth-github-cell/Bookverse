# 📚 BOOKVERSE

BOOKVERSE is a modern web-based Library Management System built using Flask and SQLite. It provides a digital platform where users can discover, read, and manage books online. The application offers secure authentication, category-based browsing, favorites management, reading history tracking, and an administrative dashboard for efficient library management.

---

## 🚀 Features

### 👤 User Features
- User Registration and Login
- Secure Password Authentication
- Browse Books by Categories
- Search Books
- Read PDF Books Online
- Add Books to Favorites
- View Reading History
- User Profile Management

### 🛠️ Admin Features
- Admin Dashboard
- Add New Books
- Update Book Information
- Delete Books
- Manage Categories
- Manage Users
- Upload PDF Files

### 🔒 Security Features
- Password Hashing
- Session Management
- User Authentication and Authorization
- Secure Access Control

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/BOOKVERSE.git
cd BOOKVERSE
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

### 6. Open Browser

```text
http://127.0.0.1:5000
```

## 📖 How BOOKVERSE Works

1. Users create an account and log in.
2. Browse books from different categories.
3. Read uploaded PDF books directly from the platform.
4. Save favorite books for quick access.
5. Track reading history.
6. Administrators manage books, categories, and users through the admin panel.
